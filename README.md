#  Password Cracker (Dictionary Attack)

##  Description
This project demonstrates a simple dictionary attack using Python to crack weak passwords.

##  Features
- Uses a wordlist (passwords.txt)
- Tries passwords one by one
- Displays cracking process
- Stops when password is found

##  How to Run
python3 cracker.py

Enter password when prompted.

##  Example Output
Shows attempts like:
Trying: 123456
Trying: password

Password found: password

##  Learning Outcome
- Learned how dictionary attacks work
- Understood password vulnerabilities
- Python file handling and loops
